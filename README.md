# Kevin Portfolio (GitHub Pages)

This is a simple, fast, single-page portfolio site you can host for free using **GitHub Pages**.

## 1) Put these files in a GitHub repo

Create a repo named:

`<your-username>.github.io`

Upload everything in this folder to the **root** of that repo, including the PDF:

- `index.html`
- `styles.css`
- `data.js`
- `script.js`
- `KV-Portfolio-2025.pdf`  ← keep the same filename
- `assets/` (optional icons)

## 2) Enable GitHub Pages

Repo → **Settings** → **Pages** → **Build and deployment**

- Source: **Deploy from a branch**
- Branch: **main**
- Folder: **/(root)**

## 3) Visit your site

`https://<your-username>.github.io`

## Customize

- Edit `data.js`
  - Add GitHub/LinkedIn in `SOCIAL`
  - Add/edit projects in `PROJECTS`
- Update the text in `index.html` for your personal summary
